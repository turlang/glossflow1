# GlossFlow v9.0 — Fase 1: Agenda Enterprise

Esta versão evolui a agenda do GlossFlow para um módulo operacional mais próximo de produtos SaaS profissionais.

## Principais melhorias

- Visualização por **Dia**, **Semana**, **Mês** e **Profissionais**.
- Agenda por profissional lado a lado.
- KPIs de agenda: agendamentos, ocupação estimada e potencial financeiro.
- Cores automáticas por serviço.
- Reagendamento visual por **drag and drop**.
- Rota administrativa para atualizar agendamentos com validação de conflito.
- CSS específico para a agenda enterprise, mantendo responsividade.

## Como usar

1. Acesse o painel administrativo.
2. Entre no módulo **Agenda**.
3. Escolha a visualização desejada: Dia, Semana, Mês ou Profissionais.
4. Para reagendar, arraste um card de atendimento para outro horário ou profissional.

## Observação técnica

O drag and drop atual usa HTML5 Drag and Drop nativo para evitar dependência extra. Em uma fase futura, pode ser substituído por bibliotecas como `dnd-kit` para interações ainda mais avançadas.
